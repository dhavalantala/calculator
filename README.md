# 🧮 Calculator

A clean, responsive web calculator built with HTML, CSS, and JavaScript.

🔗 **Live Demo:** [dhavalantala.github.io/calculator](https://dhavalantala.github.io/calculator/)

---

## ✨ Features

- Basic arithmetic operations — addition, subtraction, multiplication, and division
- Percentage (%) calculation
- Decimal point support
- Double zero (`00`) input for quick number entry
- **AC** (All Clear) to reset the display
- **DEL** to delete the last entered digit
- Responsive and minimal UI

---

## 🛠️ Built With

| Technology | Purpose |
|---|---|
| HTML | Structure & layout |
| CSS | Styling & responsiveness |
| JavaScript | Calculator logic & interactivity |

## 🚀 Getting Started

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/dhavalantala/calculator.git
   ```

2. **Navigate into the project folder**
   ```bash
   cd calculator
   ```

3. **Open `index.html` in your browser**
   ```bash
   open index.html
   ```
   Or simply double-click `index.html` in your file explorer.

No dependencies or build tools required — it runs entirely in the browser.
